Build mode is exactly what it sounds like, the classic minecraft behavior. This world, however serves a different purpose, world alteration is not the focus. However, I do want to have it because it feels like an obligated thing to offer in a voxel world.

## How it will work
Instead of a setting, it will be a game mechanic. We will add the 'God Block'. The God Block is a touchable block, and unique - there is one spawned at a random place in the world.

### Block characteristic
- When the player touches the block,
- When the crosshair goes over the God Block, a subtle label will hover over the block saying 'Touch to toggle God Mode'
- The Block becomes interactable within 2 blocks distance.
- Within 10 blocks distance, the block will emit a glow to indicate there is a special something about it

### God Mode
- An inventory appears on the bottom, with 10 squares to hold stacks of 64 blocks
- The user hand appears, holding the block selected
- Use the arrow keys left and right to select a block to change
- Block placing follows the classic minecraft flow: crosshair aim, click to place, right click to collect.
