# Feedback
This file contains various small feedback and bugfix pointers.

## Pause screen music adjustment
When the world is paused, fade out the background music while the HUD is active.

## ESC control to close screen
When the world is paused, the user should be able to hit ESC to stop the screen again.

## Default audio levels
By default, the audio levels should be set to 20% for music, 60% for SFX

## Extended audio controls
In the main hub tab, add the following controls: Next, Prev, and Play/Pause. This will give the user a quick way to toggle the music or skip to the next/previous track.

## Splashing sound bug
The splashing sound now fires everytime the user touches water - also when in the water. To make this more natural, only play the sound if the player entered the water, not while traversing through the water.
